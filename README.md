# CreatorOS

CreatorOS is an agentic growth system that turns your offer into a repeatable content-to-calls machine. This repository tracks the CreatorOS MVP (phases, contracts, specs) and serves as the canonical implementation workspace.

## Contents
- `CREATOR_OS_SPEC.md` — Product definition, ICP, value props, workflow spec.
- `PHASE_PLAN.md` — Phase-by-phase execution plan with exit criteria.
- `PROMPT_CONTRACTS.md` — Strict JSON contracts for each agent.
- `DONE_CHECKLIST.md` — Ship gate checklist for every phase.

## Getting started
1. Read the spec files above to understand each component and contract.  
2. Use the phase plan to move through onboarding → content → conversion → analytics.  
3. Commit each phase once acceptance criteria are met and the DONE checklist is satisfied.
