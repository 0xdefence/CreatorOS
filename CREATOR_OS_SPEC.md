## 9) Safety + compliance constraints
- no guaranteed-income language
- claim risk checker for unverifiable outcomes
- disclosure insertion for finance/business claims

## 10) MVP scope
### Included
- onboarding wizard
- pillar + plan generation
- content + repurpose generator
- manual publish workflow
- DM qualifier script generator
- weekly report

### Excluded (v2)
- full autoposting
- full CRM sync
- advanced attribution
- multi-user collaboration

## 11) Acceptance criteria
- user can onboard and get 30-day plan in <10 minutes
- user can generate 1 idea -> 4 channel assets
- user can run DM qualifier and produce call brief
- weekly report includes >=3 actionable optimizations
- safety checker blocks high-risk claims

- “CreatorOS turns your offer into a repeatable content-to-calls system.”
